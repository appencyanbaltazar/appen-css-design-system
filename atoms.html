Atoms

- an atom refers to the smallest and simplest component of a user interface. It represents an individual element or UI component that cannot be broken down any further without losing its functionality or meaning. Atoms are the building blocks of more complex components and UI elements.

Buttons

Primary buttons

Main call to action. They should appear just once on a page/modal/form/panel.

Large / Default

<button class ="button button--primary button--large"> Button Label </button>

Large / Disabled

<button class ="button button--primary-disabled button--large"> Button Label </button>

Large / Hover

<button class ="button button--primary-hover button--large"> Button Label </button>

Large / Focus / Active

<button class ="button button--primary-focus-active button--large"> Button Label </button>

Large / With Icon

<button class ="button button--primary button--large"> <i class ="fa-solid fa-check"> Button Label </button>
